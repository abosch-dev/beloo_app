# Beloo Test App
Aplicación para registrar localizaciones de ciclistas, ubicar a los ciclistas cercanos y poder reconstruir rutas de un ciclista

### Cosas a tener en cuenta
Aunque hay autenticación a través de JWT no se ha realizado la parte de la gestión de usuario, es decir, se comprueba con un usuario fake siempre existente.
El proceso correcto si es hacer un repositorio donde registrar usuario y donde ir a comprobar si con el token dado existe algún usuario en BD.

Por otro lado, indicar que sí se puede disponer de la documentación de la api a través del swagger.
